# react-js-vector-icons
An adaptation of react-native-vector-icons for reactjs

## Installation
 - Install npm package

```sh
npm i -S react-js-vector-icons
```

## Usage


```javascript
import Icon, { FontAwesome, Feather } from 'react-js-vector-icons';
```
```html
<Icon 
  name='tools'
  font='Entypo'
  color='green'
  size={20}
  // style={{}}
/>

<FontAwesome 
  name='bell'
  color='white'
  size={30}
  // style={{}}
/>

<Feather 
  name='award'
  color='#007ACC'
  size={40}
  // style={{}}
/>
```
